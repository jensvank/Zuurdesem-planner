Hoe de applicatie werkt
Stap 1 — Stel de starttijd in : Kies wanneer je de starter uit de koelkast haalt via de datum/tijdpicker. Standaard staat dit op 08:00 van vandaag.
Stap 2 — Markeer je drukke uren: Klik op de blauwe uurvakjes in de zijbalk om aan te geven wanneer je bezet bent (ze worden rood). Je kunt dit voor 4 dagen invullen.
Stap 3 — Bereken het schema: Klik op “Bereken schema”. De app laat dan zien:
•	Een statuskaart bovenaan: groen = geen conflicten, rood = conflicten met je agenda
•	Een visuele tijdlijn met alle stappen ingekleurd per fase (scroll horizontaal)
•	Een stap-voor-stap lijst met exacte tijden voor elke stap
Bijzondere stappen in de planning:
•	De starter (stap 1) en het koelkast rijzen (stap 11) zijn flexibele stappen — die overlappen niet automatisch met bezette tijden omdat ze sowieso “zichzelf doen”
•	Het koelkast rijzen is ingesteld op 8 uur maar kan worden verlengd tot 24 uur — handig als er conflicten zijn vlak voor het bakken
